# Mac使用笔记：Command+Tab 也能打开关闭的窗口

作为 Mac 快捷键重度用户，`Command+Tab` 是切换应用的核心操作，但它有个影响体验的痛点，你大概率也遇到过。

用 `Command+W` 关闭应用窗口后，`Command+Tab` 列表中该应用仍会保留，选中后无法直接打开，必须再按一次 `Tab` 才能切换到目标应用，十分繁琐。

通过 `Command+H` 隐藏窗口方案虽能解决，但按键位置不适合单手操作，远不如 `Command+W` 顺手。

最近发现一个 `Command+Tab` 隐藏技巧，完美解决了这个问题，亲测实用，分享给有同样困扰的用户。

**核心操作步骤**：通过 `Command+Tab` 选中已关闭窗口的应用后，保持 `Command` 键按住不松手，先松开 `Tab` 键，紧接着按下 `Option` 键，再松开 `Command` 键，最后松开 `Option` 键。完成后，已关闭窗口的应用会直接重新打开，无需多按一次`Tab`。

**更简单的单手操作法**：选中目标应用后，按住 `Command` 键的大拇指直接滑向 `Option` 键并按住，随后松开大拇指（同步松开两个按键）。完整流程为“按住 `Command`→ 选中应用 → 拇指滑按 `Option`→ 松开双键”，练习几次就能形成肌肉记忆。

掌握这个小技巧后，`Command+Tab` 的切换效率会大幅提升，能显著优化 Mac 的使用体验。
