# Mac使用笔记：解决Command+Tab痛点——单手复活关闭应用技巧

作为Mac快捷键重度用户，Command+Tab是切换应用的核心操作，但它有个影响体验的痛点，你大概率也遇到过。

用Command+W关闭应用窗口后，Command+Tab列表中该应用仍存在，选中后无法打开，必须再Tab一次才能切换到目标应用，十分繁琐。

网传的Command+H隐藏窗口方案虽能解决，但按键位置不适合单手操作，远不如Command+W顺手，多数人仍会被迫接受切换麻烦。

最近发现一个Command+Tab隐藏技巧，完美解决此问题，亲测实用，分享给有同样困扰的用户。

核心操作：Command+Tab选中已关闭窗口的应用后，保持Command按住，松开Tab，紧接着按Option，再松开Command，最后松开Option。此时，关闭的应用会直接重新打开，无需多Tab一次。

更简单的单手操作：选中应用后，按住Command的大拇指直接滑到Option并按住，再松开大拇指（同步松开两键）。流程为“按住Command→选中应用→拇指滑按Option→松开双键”，几次就能形成肌肉记忆。

掌握这个技巧后，Command+Tab切换效率大幅提升，能显著优化使用体验。
